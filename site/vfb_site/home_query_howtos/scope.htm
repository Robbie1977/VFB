How search scope works

You can limit the scope of anatomy terms that are found in the site search to 'All Neuroanatomy' or 'Adult Brain'.
Only the terms found in the given scope will be displayed in the autocomplete boxes, meaning all the subsequent queries will only contain results from the chosen scope.
Please note:

  • The default scope is the Adult Brain.
  • You can change the scope for each search you run, from the homepage or the 'Anatomy/Neuron Finder' page. The new scope will then apply for the subsequent queries you do (neuron classes, individual neurons, expression, etc).

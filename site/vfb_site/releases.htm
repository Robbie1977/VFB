Release 2013/08/29

Feature  Stage   Status note 
Updated Chado Database and Ontology v.16/07/13 Live This release includes the most recent FlyBase Chado DB version (16/07/13) and correspondingly updated anatomy ontology, which means more results for the VFB queries.
Third party datasets with 3D stacks Live We have released several third party datasets complete with registered third party 3D stacks. Integrating new reasoner for classes Live The new ELK-based reasoner wrapper Brain is integrated to replace the previous JFact-based reasoner. The new reasoner is an order of magnitude faster than the previous one. The new reasoner reduced the time of execution of the longest expression queries from about 15 sec to just under 1 sec.

Feature  Stage   Status note

2. New database views implemented for compatibility with the latest FlyBase releases resulted in increased query speed.
New query result flagging mechanism Live The limit of resolution for expression and phenotype data recorded in FlyBase is a single cell. When you query a structure for expression or phenotypes, we return records of expression or phenotypes in cells that overlap the structure. As there is a danger that the gene or transgene products or phenotypes in question may be localised to some part of the cell that does not overlap the queries structure, we flag such records with a warning. If you download the query results as a csv file, this flag can be used to filter or sort them.
